## 🚀 Como Executar Localmente

```bash
# Clone o repositório
git clone https://github.com/karinab020/tts-app.git

# Acesse a pasta do projeto
cd tts-app

# Execute (não requer instalação)
open index.html  # Mac
start index.html # Windows
xdg-open index.html # Linux
